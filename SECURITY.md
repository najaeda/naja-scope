# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability in naja-scope, please
report it privately so we can address it before public disclosure.

**Email:** [contact@keplertech.io](mailto:contact@keplertech.io)

Please include:

- a description of the issue and its potential impact,
- steps to reproduce (a minimal example helps),
- the `naja-scope` and `najaeda` versions affected.

We aim to acknowledge reports within a few business days and will keep you
informed as we work on a fix. Please do not open a public issue for security
reports.

## Supported versions

naja-scope is in active development; security fixes are made against the latest
released version on PyPI.
